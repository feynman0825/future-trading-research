# future-trading-research

## 研究目的

1. 研究中国期货市场在1m钟级别的机器学习算法以及交易策略
2. 构建期货数据实时采集系统
3. 构建期货实时交易系统

## 算法部分

1. 研究对象包含上海期货交易所的所有标的
2. 只研究1m级别或tick级别的算法和交易策略
3. 历史行情数据来源可以参考JQuant
4. label采用上下夹板方式标注

#### EDA方向

* 关注在分钟级别，e.g 1-30m 的收益率分布、lebel分布
* 日内周期的分布规律
* 股指、国债期货的联动规律
* 自相关性
* 特征、因子的feature importance
* 价格突破的规律
